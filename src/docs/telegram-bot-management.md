# صفحه مدیریت ربات تلگرام

## مسیر

`/dashboard/telegram`

## کاربرد

این صفحه کنترل عملیاتی ربات را در پنل فراهم می‌کند و نیاز به بررسی مستقیم سرور یا BotFather برای عملیات روزمره را کاهش می‌دهد.

## امکانات

- نمایش هویت ربات و Username
- نمایش URL وبهوک، درخواست‌های معلق و آخرین خطای Telegram
- تنظیم یا بروزرسانی وبهوک با آدرس عمومی HTTPS
- حذف وبهوک
- ارسال پیام آزمایشی به کاربر متصل
- نمایش تعداد کاربران متصل و بدون اتصال
- جستجو و فیلتر کاربران بر اساس وضعیت اتصال
- حذف اتصال تلگرام کاربر و پاک‌سازی sessionهای فعال او
- نمایش تعداد وظایف، گزارش‌ها و فایل‌های ثبت‌شده از ربات
- نمایش پروژه‌های دارای تخصیص ناقص و وظایف عقب‌افتاده
- نمایش دستورات فعال ربات
- نمایش آخرین فعالیت‌های ثبت‌شده از منبع `telegram_bot`

## ارتباط با فرآیند جدید Excel

Excel فقط اطلاعات پروژه، فاز، زمان‌بندی و مالی را وارد می‌کند. افراد و مسئولیت‌ها داخل پنل تعیین می‌شوند. ربات در زمان ثبت وظیفه فقط افراد واقعی پروژه را نمایش می‌دهد؛ بنابراین پروژه واردشده تا قبل از تکمیل تخصیص نیروی انسانی برای ثبت وظیفه آماده نیست.

## اتصال امن کاربران

ورود دستی `Telegram User ID` و `Chat ID` مسیر اصلی اتصال نیست. مدیر از جدول کاربران دکمه «ساخت کد اتصال» را می‌زند. پنل یک کد ۸ کاراکتری و Deep Link ربات نمایش می‌دهد. کاربر با بازکردن لینک یا ارسال `/link CODE` حساب خود را متصل می‌کند.

- کد فقط ۱۵ دقیقه معتبر است.
- کد یک‌بار مصرف است.
- شناسه‌ها از Update رسمی Telegram دریافت می‌شوند.
- اتصال تکراری یک حساب تلگرام به چند کاربر مسدود است.
- برای اتصال مجدد می‌توان کد جدید ساخت؛ برای قطع کامل اتصال از «حذف اتصال» استفاده می‌شود.
