# ورود پروژه و فاز از اکسل در فرانت‌اند

## ساختار جدید

فایل اکسل فقط شامل اطلاعات پروژه، فاز، زمان‌بندی و مالی ساده است. اطلاعات افراد و مسئولیت‌ها در Excel ثبت نمی‌شود.

- شیت `Projects`: عنوان، توضیحات، وضعیت، اولویت و تاریخ‌های پروژه
- شیت `Phases`: عنوان و ترتیب فاز، تاریخ‌ها و مبالغ مالی ساده

## جریان تخصیص افراد

پس از ورود موفق، پروژه با وضعیت «نیازمند تخصیص» نمایش داده می‌شود. مدیر از صفحه ویرایش پروژه موارد زیر را تعیین می‌کند:

- مسئول پروژه
- مسئول یا مسئولان هر فاز
- اعضای پروژه

پس از ذخیره تخصیص‌ها، نقش دقیق هر عضو از تب «اعضا» در صفحه جزئیات پروژه قابل تنظیم است.

## تغییرات رابط کاربری

- متن‌ها و راهنمای Modal ورود اکسل با قرارداد جدید هماهنگ شده‌اند.
- نتیجه ورود، تعداد ردیف‌های پروژه و فاز را جداگانه نمایش می‌دهد.
- خطاها نام شیت و شماره ردیف را نشان می‌دهند.
- برای هر پروژه ایجادشده، لینک مستقیم تکمیل افراد و مسئولیت‌ها وجود دارد.
- پروژه‌های بدون مسئول یا عضو در فهرست و صفحه جزئیات با هشدار مشخص می‌شوند.
