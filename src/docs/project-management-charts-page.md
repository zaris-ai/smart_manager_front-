# صفحه نمودارهای مدیریتی پروژه‌محور

## هدف

یک صفحه مستقل برای گزارش‌های مدیریتی پروژه‌ها اضافه شد تا تمام نمودارها بر اساس خود پروژه‌ها نمایش داده شوند، نه صرفاً وضعیت‌های تجمیعی.

مسیر صفحه:

```txt
/dashboard/project-charts
```

## محتوای صفحه

این صفحه برای هر پروژه یک ردیف داده تحلیلی می‌سازد و نمودارها را بر همان اساس نمایش می‌دهد.

### نمودارهای اضافه‌شده

1. **وضعیت همه پروژه‌ها بر اساس درصد پیشرفت**
   - هر پروژه یک ستون افقی دارد.
   - طول ستون، درصد پیشرفت پروژه را نشان می‌دهد.
   - رنگ ستون، وضعیت مدیریتی پروژه را مشخص می‌کند.

2. **وظایف هر پروژه**
   - تعداد وظایف هر پروژه به تفکیک وضعیت نمایش داده می‌شود.
   - وضعیت‌های پشتیبانی‌شده: برای انجام، در حال انجام، مسدود، انجام‌شده.

3. **ریسک زمانی هر پروژه**
   - برای هر پروژه یک امتیاز ریسک از ۰ تا ۱۰۰ محاسبه می‌شود.
   - تاخیر پروژه، نزدیکی سررسید و وظایف عقب‌افتاده در این امتیاز لحاظ شده‌اند.

4. **گزارش‌ها و فایل‌ها بر اساس پروژه**
   - تعداد گزارش کار، کل فایل‌ها و فایل‌های گزارش برای هر پروژه نمایش داده می‌شود.

5. **فازهای هر پروژه**
   - تعداد فازها و مجموع روزهای برنامه‌ریزی‌شده فازها برای هر پروژه نمایش داده می‌شود.

6. **جدول مدیریتی پروژه‌ها**
   - همان داده‌های اصلی نمودارها در قالب جدول مدیریتی نمایش داده می‌شود.

## منطق رنگ‌بندی وضعیت پروژه‌ها

- سبز: سالم یا تکمیل‌شده
- آبی: فعال
- نارنجی: نزدیک سررسید
- قرمز: عقب‌افتاده
- خاکستری: متوقف یا لغوشده

## جلوگیری از تکرار نمودارها

- `/dashboard` برای عملیات روزانه باقی می‌ماند.
- `/dashboard/project-overview` برای تحلیل‌های تجمیعی پورتفو باقی می‌ماند.
- `/dashboard/project-charts` برای نمودارهای مقایسه‌ای پروژه‌محور اضافه شد.
