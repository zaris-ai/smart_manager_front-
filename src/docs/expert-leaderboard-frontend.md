# Expert leaderboard frontend

Manager route: `/dashboard/expert-leaderboard`

The page contains:

- 7, 30, and 90-day presets plus a custom date range
- Top-three competitive podium
- Most-active and least-active expert cards
- Rank movement against the preceding equivalent period
- Top-ten score chart
- Full expert ranking with duration, active days, entries, project count, deliverable rate, progress, and last activity
- Links to the existing expert work-detail page

The management dashboard also displays the current 30-day most-active and least-active experts. Both the page and sidebar entry are restricted to manager and board roles.
