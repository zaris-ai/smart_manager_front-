# صفحه درخواست مرخصی

مسیر پنل: `/dashboard/leave-requests`

- برای مدیر و کارشناس در سایدبار نمایش داده می‌شود.
- هر دو نقش می‌توانند درخواست روز کامل، نیم‌روز یا ساعتی ثبت کنند.
- مدیر تب جداگانه «کارتابل بررسی» دارد و می‌تواند درخواست سایر کاربران را تأیید یا رد کند.
- مدیر نمی‌تواند درخواست خودش را بررسی کند.
- درخواست در وضعیت `pending` قابل ویرایش یا لغو است.
- تاریخ‌ها در رابط کاربری شمسی و در API به‌صورت Gregorian `YYYY-MM-DD` ارسال می‌شوند.
