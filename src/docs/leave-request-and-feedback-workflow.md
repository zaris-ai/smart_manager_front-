# فرآیند مستقل درخواست مرخصی و انتقادات و پیشنهادها

## صفحات

- `/dashboard/leave-requests/new`: صفحه کامل ثبت درخواست مرخصی برای مدیر و کارشناس.
- `/dashboard/leave-requests`: سوابق شخصی و کارتابل بررسی مدیران؛ ایجاد درخواست از این صفحه به مسیر جدید هدایت می‌شود.
- `/dashboard/feedback`: ثبت، پیگیری و پاسخ‌دهی به انتقادها و پیشنهادها.

## دسترسی

هر دو نقش مدیر و کارشناس می‌توانند درخواست مرخصی و بازخورد ثبت کنند. کارتابل بررسی مرخصی و بازخورد فقط برای مدیران فعال است. مدیر نمی‌تواند درخواست مرخصی یا بازخورد خودش را در کارتابل مدیریت بررسی کند.

## حریم خصوصی بازخورد

در ارسال ناشناس، نام ارسال‌کننده در پاسخ API مدیریتی مخفی می‌شود. شناسه واقعی برای ممیزی امنیتی در پایگاه داده نگهداری می‌شود و برای سایر کاربران نمایش داده نمی‌شود.
